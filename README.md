# Adaptive_Control_SC617
Course taken in Autumn 2026 on Adaptive Control Theory.

## Homework 1

Theoretical Questions related to some concepts on adaptive control taught during the lectures.

## Homework 2

Implementing a dual loop feedback controller for a drone from a research paper.

## Homework 3

Summarizing a research paper that suggests an adaptive controller for a Quadcopter.

## Homework 4

Attitude only Adaptive controller for a solid body (for eg. a satellite).

## Final Project

Implementing the adaptive controller for a quadcopter encountered in homework 3.
